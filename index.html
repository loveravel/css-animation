Что такое анимация?

Анимация (от фр. animation «оживление; одушевление»)
технология мультимедиа; воспроизведение последовательности картинок, создающее впечатление движущегося изображения

  • Много различного софта

  • ...

Однако и CSS позволяет создавать простые анимации без использования JavaScript.

Трансформации

CSS-свойство transform позволяет поворачивать, маштабировать, наклонять или сдвигать элемент.

  • matrix (матрица)
  • translate (перенос)
  • scale (масштаб)
  • rotate (поворот)
  • skew (наклон)

Timing Function

Обозначает математическую функцию, которая описывает, как быстро изменяются одномерные значения во время анимации.

Позволяет изменять скорость анимации в течение ее продолжительности.

  • linear (движется от начала к концу с постоянной скоростью)
  • ease (начинается медленно, резко ускоряется, а затем постепенно замедляется к концу)
  • ease-in (начинается медленно, а затем постепенно ускоряется до конца, после чего она резко останавливается)
  • ease-in-out (начинается медленно, ускоряется, а затем замедляется к концу)
  • ease-out (начинается внезапно, а затем постепенно замедляется к концу)
  • cubic-bezier(x1, y1, x2, y2)
  • steps(n, <direction>)

Плавные переходы

Свойство CSS transition — сокращенное свойство для transition-property, transition-duration, transition-timing-function, и transition-delay.

Позволяет определять переходное состояние между двумя состояниями элемента.

Различные состояния могут быть определены с помощью псевдоклассов, таких как :hover и :active или установлены динамически с помощью JavaScript.

Покадровое управление анимцией

Для этого понадобятся:

  • Раскадровка (@keyframes)
  • CSS-свойтво animation

@keyframes

Позволяет управлять промежуточными шагами путем определения стилей для ключевых кадров (или точек) вдоль последовательности анимации.

  • from
  • nn%
  • to

animation

  • animation-name (имя раскадровки)
  • animation-duration (длительность)
  • animation-timing-function
  • animation-delay (задержка перед воспроизведением)
  • animation-iteration-count (кол-во итераций): infinite | <number>
  • animation-direction (направление вопроизведения раскадровки): normal | reverse | alternate | alternate-reverse
  • animation-fill-mode (применение стилей к объекту до и после выполнения анимации): none | forwards | backwards | both
  • animation-play-state (состояние воспроизведения): running | paused

Что можно анимировать

Так как некоторые свойства не имеет смысла анимировать, перечень доступных для анимирования свойств ограничен определенным набором.

+/-

  • Простые анимации делаются просто
  • Быстрые и не создают нагрузку на CPU
  • JavaScript-анимации более гибкие. В них может присутствовать любая анимационная логика
  • Можно изменять не только свойства, а также создавать новые элементы с помощью JavaScript для анимации
